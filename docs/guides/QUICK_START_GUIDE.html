✅ Schritt 1: Voraussetzungen prüfen

Stelle sicher, dass folgende Tools installiert sind:

Git (Version 2.0+)
git --version
Node.js (Version 18+) und npm
node --version && npm --version
Java 17+ (für Backend)
java --version
Docker (für Datenbank & Keycloak)
docker --version

💡 Tipp

Falls etwas fehlt, findest du hier die Download-Links:

  • Git installieren
  • Node.js installieren
  • Java (Temurin) installieren
  • Docker Desktop installieren

📦 Schritt 2: Repository klonen

GitHub Repository klonen
git clone https://github.com/freshfoodz/freshplan-sales-tool.git cd freshplan-sales-tool
Zum richtigen Branch wechseln
git checkout main git pull origin main

Projekt-Struktur

frontend/
↔
backend/
↔
infrastructure/

⚙️ Schritt 3: Backend starten

Docker Services starten
cd infrastructure docker-compose up -d

Startet PostgreSQL und Keycloak automatisch

Quarkus Backend starten
cd ../backend ./mvnw quarkus:dev
PostgreSQL lokal installieren

Erstelle eine Datenbank "freshplan" auf Port 5432

Keycloak separat aufsetzen

Siehe Keycloak Setup Guide

✅ Backend läuft wenn:

  • Console zeigt: "Listening on: http://localhost:8080"
  • http://localhost:8080/api/ping antwortet

🎨 Schritt 4: Frontend starten

Dependencies installieren
cd frontend npm install
Development Server starten
npm run dev

✅ Frontend läuft wenn:

  • Console zeigt: "VITE ready in X ms"
  • http://localhost:5173 zeigt die App

🎉 Geschafft!

FreshPlan läuft jetzt lokal auf deinem Rechner!

Wichtige URLs:

  • Frontend: http://localhost:5173
  • Backend API: http://localhost:8080/api
  • Keycloak Admin: http://localhost:8081
