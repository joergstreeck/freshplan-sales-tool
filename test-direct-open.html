FreshPlan - Direct File Open Test

Behebung des Problems

Problem:

Beim direkten Öffnen der index.html Datei (file:// Protokoll) können ES6 Module aus Sicherheitsgründen nicht geladen werden.

Lösungen:

  1. Empfohlen: Lokalen Server verwenden 
    cd /Users/joergstreeck/freshplan-sales-tool
npm run dev

    Dann öffnen Sie: http://localhost:5173

  2. Alternative: Build-Version verwenden 
    cd /Users/joergstreeck/freshplan-sales-tool
npm run build
npm run preview

    Dann öffnen Sie: http://localhost:4173

  3. Für Produktion: Auf Webserver hochladen

    Laden Sie den Inhalt des dist/ Ordners auf einen Webserver hoch.
